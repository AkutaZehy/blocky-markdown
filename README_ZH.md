# Blocky Markdown 使用说明

一个具有 WordPress 风格块状编辑体验的本地 Markdown 编辑器，采用 Y2K 复古设计风格。

## 快速开始

1. 用浏览器打开 `index.html` 文件即可使用
2. 无需安装任何框架或构建工具
3. 可以导入 `example.md` 文件来快速体验所有功能

## 主要功能

### 块类型
- 📝 **段落** - 普通文本内容
- 📌 **标题** - H1-H6 级别标题，可选择级别
- 📋 **列表** - 有序或无序列表
- 💻 **代码** - 代码块，支持语言标识
- 📊 **表格** - 可视化表格编辑器
- 🔗 **链接** - 超链接编辑器，支持文本、URL 和标题
- 💬 **引用** - 块引用

### 编辑功能
- **添加块** - 点击底部的块类型按钮
- **编辑内容** - 直接在文本框中输入
- **表格编辑** - 点击 ✏️ 按钮打开可视化编辑器
- **链接编辑** - 点击 ✏️ 按钮打开链接编辑器
- **移动块** - 使用 ↑↓ 按钮调整块的顺序
- **删除块** - 点击 🗑️ 按钮

### 导入/导出
- **📥 导入 MD** - 粘贴 Markdown 内容，自动解析为块
- **📤 导出 MD** - 导出为 Markdown 格式
- **📋 复制** - 复制到剪贴板
- **💾 下载** - 下载为 .md 文件

### 自动保存
- 所有编辑会自动保存到浏览器的 localStorage
- 重新打开页面时会自动加载之前的内容
- 点击"清空所有"可以清除所有块

## Y2K 设计风格

- 🌈 霓虹色彩和渐变效果
- ✨ 发光阴影和边框
- 🎨 复古的色彩搭配
- 💫 流畅的动画效果
- 🎯 直观的交互界面

## 技术特点

- ✅ 纯 HTML + CSS + JavaScript
- ✅ 无需任何框架（Vue/React/Astro/Svelte）
- ✅ 无需构建工具
- ✅ 支持 Marked.js 库（含备用解析器）
- ✅ 本地运行，数据存储在浏览器中
- ✅ 响应式设计，支持移动端

## 浏览器兼容性

支持所有现代浏览器：
- Chrome / Edge
- Firefox
- Safari
- Opera

需要支持 ES6、CSS Grid/Flexbox 和 LocalStorage

## 使用提示

1. **表格编辑器**
   - 设置行数和列数
   - 点击"生成"创建表格
   - 在单元格中直接输入内容
   - 点击"确认"插入到文档中

2. **链接编辑器**
   - 输入链接文本（显示的文字）
   - 输入 URL 地址
   - 可选：输入链接标题（鼠标悬停时显示）
   - 点击"确认"插入链接

3. **代码块**
   - 输入编程语言名称（如 javascript）
   - 在文本框中输入代码
   - 系统会自动添加代码块标记

4. **导入功能**
   - 支持标准 Markdown 语法
   - 自动识别并分割为不同的块
   - 保留原有格式

开始享受你的复古 Markdown 编辑体验吧！ 🎨✨
