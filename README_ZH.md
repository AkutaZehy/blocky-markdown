# Blocky Markdown 使用说明

一个极简主义的、无需框架的 Markdown 编辑器，提供类似 WordPress 的块状编辑体验。

## 主要功能

- 🎨 **双主题系统** - 极简主义（默认）或 Frutiger Aero 美学风格
- 📝 **块状编辑** - WordPress 风格的块系统组织内容
- 📊 **增强表格编辑器** - 网格导航 + 专用单元格内容编辑器的分屏视图
- 🔗 **智能内容管理** - 大纲侧边栏和缓存区提供更好的组织方式
- 💾 **导入/导出** - 支持前置数据的 Markdown 文件读写
- 🔄 **自动保存** - 自动保存到浏览器 localStorage
- 🚀 **无需构建** - 纯 HTML、CSS 和 JavaScript

## 主题系统

### 极简主义主题（默认）
- 黑白配色方案
- 等宽字体，清晰易读
- 无装饰元素
- 专注写作的体验

### Frutiger Aero 主题
- 现代蓝色调色板
- 柔和的渐变和阴影
- 专业、精致的界面

点击页眉的 🎨 Theme 按钮切换主题。

## 支持的块类型

- ⚙️ **Frontmatter** - Jekyll/Hugo YAML 前置数据
- 📝 **段落** - 普通文本内容
- 📌 **标题** - H1-H6 标题，可选择级别
- 📋 **列表** - 有序或无序列表
- 💻 **代码** - 带语言标签的代码块
- 📊 **表格** - 可视化交互表格编辑器
- 💬 **引用** - 块引用
- ➖ **HR** - 水平分隔线
- ⏎ **BR** - 换行符
- 🌐 **HTML** - 原始 HTML 块
- 📊 **Mermaid** - Mermaid 图表支持

## 增强功能

### 大纲侧边栏
- 实时显示文档结构
- 点击任意项跳转到对应块
- 显示块类型图标和预览文本
- 随编辑自动更新

### 缓存区
- 拖动块到缓存区临时存储
- 比直接删除更安全
- 可在缓存区或工作区编辑块
- 永久删除仅在缓存区可用

### 表格编辑器改进
- **分屏视图**：网格导航（左）+ 内容编辑器（右）
- **添加/删除行**：动态行管理，不丢失数据
- **添加/删除列**：随时调整表格宽度
- **单元格选择**：点击单元格舒适地编辑长内容
- **链接支持**：在表格单元格中轻松编辑 Markdown 链接

## 使用方法

1. **打开编辑器**
   - 在浏览器中打开 `index.html`
   - 无需服务器或构建过程

2. **选择主题**
   - 点击 🎨 Theme 在极简主义和 Frutiger Aero 风格间切换
   - 选择会自动保存

3. **添加块**
   - 点击工作区底部的块类型按钮
   - 使用大纲侧边栏浏览文档

4. **编辑内容**
   - 直接在文本块中输入
   - 表格使用 ✏️ 按钮打开增强编辑器
   - 拖动块或使用 ↑↓ 按钮重新排序

5. **管理块**
   - 拖动块到缓存区（📦 按钮）
   - 需要时拖回工作区
   - 仅可从缓存区永久删除

6. **导入/导出**
   - 点击"📥 Import"粘贴现有 Markdown
   - 点击"📤 Export"复制或下载 Markdown
   - 支持 Jekyll frontmatter

7. **表格编辑**
   - 点击表格块的 ✏️
   - 使用 +/- Row/Col 按钮调整大小
   - 选择单元格在专用编辑器中编辑长内容
   - 点击 Confirm 保存更改

## 使用技术

- **HTML5** - 结构
- **CSS3** - 带 CSS 变量的双主题系统
- **JavaScript (ES6+)** - 应用逻辑
- **Marked.js** - Markdown 解析和渲染

## 浏览器兼容性

支持所有现代浏览器：
- ES6 JavaScript
- CSS Grid 和 Flexbox
- CSS 自定义属性
- LocalStorage API
- 拖放 API

## 设计理念

**极简优先**：默认主题优先考虑清晰和专注，仅使用黑白灰和等宽字体。完美的无干扰写作体验。

**专业替代**：Frutiger Aero 主题提供现代、精致的外观，带有微妙的颜色和渐变，适合喜欢更丰富视觉效果的用户。
